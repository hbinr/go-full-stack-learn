
- daemonize  是否以守护进程方式启动,默认为 no，推荐设置为yes
  这样在启动的时候，启动日志就会打印到我们配置的日志文件中    
-  prot      对外端口号，单机多实例需要配置端口号。默认端口为6379
-  logfile   系统日志，设置保存日志的文件名
-  dir       工作目录，如日志文件，持久化文件是存储在哪个目录。 
  注意上面的logfile指定的是一个文件名，该dir指定路径
